# Aaron Shuang (ATS3456)

This is the final component of the Tinker program, integrating all previous development phases into a complete assembler and simulator suite.

## Execution Guide

### Assembler
Convert your `.tk` assembly files into executable binary `.tko` files:

```bash
./hw5-asm <input_filename> <output_filename>
```

### Simulator

```bash
./hw5-sim <input_filename>
```
